Reflection

During this module I learned the basics of web development using HTML and CSS. At the beginning I found it difficult to understand how different HTML elements worked together to create the structure of a webpage. However, through practice I became more confident using semantic elements such as header, nav, main and footer.

One of the main challenges I faced was debugging CSS layout issues. Sometimes elements were not aligned correctly or the layout broke on smaller screens. By using CSS Grid and media queries I was able to create a responsive design that adapts to mobile devices.

For the design of my website I chose a simple colour palette with dark navigation and light backgrounds to improve readability. I was influenced by modern portfolio websites which focus on minimal design and clear navigation.

Throughout development I regularly committed my changes to GitHub which helped track my progress and manage updates to the website.
